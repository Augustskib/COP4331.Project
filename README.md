Contact Manager

A full-stack web application built as part of COP4331. The project allows users to create an account, log in securely, and manage a personal contact list through a simple web interface.

Users can add, search, edit, and delete contacts, with all data stored in a database and accessed through a custom API. The goal of the project was to gain experience building a complete web application from the frontend all the way to the database.

What it does:

After creating an account and logging in, users can:

View their saved contacts
Add new contacts
Search for existing contacts
Update contact information
Delete contacts
Log out and return later with their data preserved

Built with:
HTML
CSS
JavaScript
PHP
MySQL
REST API architecture

About the project:

This project was developed by a team of students at UCF as part of COP4331 (Processes for Object-Oriented Software Development).
The application was designed to simulate a real-world software development workflow. Development was organized through GitHub, with team members working on different parts of the stack and integrating their work into a single application.

Throughout the project we worked with frontend development, backend APIs, database design, version control, testing, and deployment.
